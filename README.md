##  Getting Started

1. Clone the repo and navigate to the project root:
    ```bash
    git clone <your-repo-url>
    cd <repo-root>/demo
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Copy the environment template and add your API key:
    ```bash
    cp .env.example .env
    # then edit:
    # OPENAI_API_KEY=sk-...
    ```

4. Launch the demo:
    ```bash
    npm run dev
    ```
   Then visit `http://localhost:3000` to chat away!
